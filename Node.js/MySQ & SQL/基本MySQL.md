## 5. 創建資料表
* ### DataType 數據類型 :
	1. int : 整數。
	2. varchar(len) : 字符串。
	3. tinyint(1) : 布林值。
* ### 字段的特殊標識 :
	1. PK(Primary key) : 主鍵，唯一標識。
	2. NN(Not Null) : 值不允許為空。
	3. UQ(Unique) : 值唯一，不允許重複的值。
	4. AI(Auto Increment) : 值自動增加。

### 5-1 點選創建
